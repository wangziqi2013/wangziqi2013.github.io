# SPEC 2006

## Unpack the Tarball

If you acquired SPEC 2006 in a tarball, the first step is to extract it into a directory:

```
mkdir SPEC2006
cd SPEC2006
tar -xvf [tarball file name]
```

In all the examples below, we assume that SPEC is extracted into directory `SPEC2006`.
